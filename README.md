# Supermarket Sales Analysis
## Project Overview
This project focuses on the analysis of a Supermarket Sales dataset to identify trends in product performance, branch sales, and customer purchasing behavior. By leveraging data visualization, the project aims to uncover actionable business insights regarding revenue generation and customer demographics.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Dataset
Supermarket Sales Dataset from Kaggle. (https://www.kaggle.com/datasets/chadwambles/supermarket-sales)

## Key Insights
- **Personal Care** was the top-selling category, generating $27,050 in total revenue and the most reward points (1,483 points), making it the most rewarding category.
- **Branch A** achieved the highest sales with a total of $82,811.64, while Branch B had the lowest. Branch A also has the highest average tax per transaction at $8 (or 8%), indicating customers there tend to spend more per purchase.
- **Chicago** generated the highest revenue among all cities, totaling $42,584.71.
- **Customer Demographics:** Male customers tend to spend more, with an average of $121.82 compared to $114.97 for female customers.
- **Membership Status:** Member customers outspent Normal customers on average, with $122.51 vs $114.40 per transaction.
- **Top Product:** Shampoo was the best-selling product with 2,238 units sold.

## Conclusion
This project analyzed supermarket sales data using Python and identified key trends in product performance, branch sales, and customer purchasing behavior. Data visualization was used to present these insights clearly, providing a better understanding of the spending patterns and demographic preferences within the dataset.
